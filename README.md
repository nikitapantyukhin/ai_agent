# 🤖 Market AI Agent

Интеллектуальная аналитическая система для предпринимателей на маркетплейсах. Агент анализирует ниши, предсказывает тренды.

## 🚀 Основные возможности
- **ИИ-аналитика**: Глубокий анализ товарных категорий с помощью моделей Llama-3 / Phi-3.
- **Трендовый радар**: Автоматическое выделение стилей, цветов и фасонов.
- **Финансовое планирование**: Расчет ROI, маржинальности и оценка рисков.
- **Интеграция с логистикой**: Советы по закупу в Китае и Турции.

## 🛠 Инструкция по запуску

### 1. Подготовка окружения
Убедитесь, что у вас установлен Python 3.10 или выше.
```bash
git clone <https://github.com/nikitapantyukhin/ai_agent.git>
cd ai_agent

# Создайте и активируйте виртуальное окружение
python3 -m venv venv
source venv/bin/activate
```

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 3. Настройка ключей
Создайте файл `.env` в корневой папке и добавьте ваш токен Hugging Face:
```text
HF_TOKEN=hf_ваш_токен_здесь
```

### 4. Запуск приложения
```bash
python3 -m streamlit run app.py
```
